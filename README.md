## 👨‍💻 About Me

I am a third-year **Management Information Systems** student at **Ankara Bilim University**, focusing on artificial intelligence, autonomous UAV systems, computer vision and simulation technologies.

I develop and test AI-supported UAV and air-defense scenarios using **Python, C/C++, ROS, Gazebo, NVIDIA Isaac Sim, PyBullet, ArduPilot SITL, OpenCV and YOLO**. My work includes object detection and tracking, dataset preparation, autonomous mission logic, multi-UAV coordination, flight-control integration and reinforcement learning.

## 🚀 Featured Projects

### 🛩️ Kutay Swarm UAV System
**ROS • Gazebo • ArduPilot SITL • YOLO • ZeroMQ**

A multi-UAV swarm and kamikaze mission scenario in which several UAVs operate in the same simulation environment.

- Developed YOLO-based target detection and integrated inference results into the mission flow.
- Built a ZeroMQ-based inference worker and swarm orchestrator for centralized target-data sharing.
- Tested autonomous missions, flight connections and multi-UAV coordination with Gazebo, ArduPilot SITL and QGroundControl.

[View Repository](https://github.com/KutayEter/Swarm-UAV-System-Gazebo)

---

### 🎯 RL-Based UAV Air Combat Simulator
**Python • PyBullet • PPO • OpenCV • FSM**

A UAV air-combat prototype featuring a PPO-trained pursuing UAV and an FSM-controlled evading UAV.

- Trained the RL policy for a total of **6 million steps**.
- Designed a 26-feature observation space, reward structure and lock-on scoring system.
- Implemented evasive maneuvers including Scissors, Split-S and break turns.
- Added OpenCV-based HUD, third-person camera and nose-camera visualizations.

[View Repository](https://github.com/KutayEter/DogFight-UAV-RL)

---

### 👁️ Kutay UAV Tracking System
**Gazebo • ArduPilot SITL • YOLO • QGroundControl**

An autonomous patrol and aerial-target tracking system developed in a simulation environment.

- Prepared and annotated a custom dataset for aerial-target detection.
- Integrated the trained YOLO model with the simulation camera stream.
- Implemented object tracking, target centering, lock-on duration and mission-termination logic.
- Tested perception, decision-making and flight-control processes end to end.

[View Repository](https://github.com/KutayEter/Gazebo-YOLO-UAV)
